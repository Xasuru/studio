# **App Name**: Mission Control

## Core Features:

- Home Screen Debrief: Displays a summary of the student's daily tasks and a calendar sync preview.
- Task List Management: Allows users to add, view, and mark tasks as complete.
- Calendar View: Presents a static calendar grid with visual highlights for important dates.
- New Task Input: Enables users to add new tasks to their task list with subject and name details.

## Style Guidelines:

- Background: Pitch Black (#000000) to provide a high-contrast, dark mode experience, promoting focus.
- Primary Action Color: Safety Orange (#FF6600) to highlight key interactive elements and actions, drawing attention in the dark interface.
- Secondary Elements: Dark Grey (#1E1E1E) for cards and containers, creating depth and separation without overwhelming contrast.
- Font: 'Inter' (sans-serif) for body text, headlines, and user interface elements, offering excellent legibility in a dark environment.
